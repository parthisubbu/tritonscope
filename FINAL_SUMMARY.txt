💡 WHAT YOU CAN DO NOW:

1. Load Triton Kernels
   $ tritonscope analyze softmax.py --kernel softmax_kernel

2. Get System Diagnostics
   $ tritonscope doctor

3. Analyze via Python API
   ```python
   from tritonscope import analyze, AnalysisConfig
   report = analyze(config)
   ```

4. Generate Reports
   - JSON: `tritonscope analyze ... --json report.json`
   - Markdown: `tritonscope analyze ... --markdown report.md`
   - Terminal: `tritonscope analyze ...` (default)

5. Run Tests
   $ pytest tests/ -v
